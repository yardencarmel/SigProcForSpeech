         Bug was: blended waveform computed but original path passed to
         preprocess_ref_audio_text() — blend was silently discarded.

[V] 4.3  Upgrade emotion transfer from waveform-blend to mel-space blend
         Approach implemented:
           a) Load both reference audios
           b) Compute log-mel spectrograms for both (matching F5-TTS/vocos params)
           c) Interpolate: mel_cond = (1-w)*mel_id + w*mel_emo
           d) Invert via vocos vocoder to temp WAV → passed to preprocess pipeline
         File: scripts/run_emotion_transfer.py  ← DONE
